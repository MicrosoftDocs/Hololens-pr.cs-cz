# <a name="hololens-2-development-edition-faq"></a>Nejčastější dotazy k verzi HoloLens 2 pro vývoj Edition

## <a name="general-questions"></a>Obecné otázky

### <a name="what-do-i-get-with-the-hololens-2-development-edition"></a>Co získám vývojovou edicí HoloLens 2?

Vývojová edice HoloLens 2 zahrnuje HoloLens 2, $500 v kreditech Azure, bezplatnou zkušební verzi Unity pro a 3 měsíce zkušebního předplatného Pixyz plug-in.

### <a name="what-is-the-difference-between-hololens-2-and-hololens-2-development-edition"></a>Jaký je rozdíl mezi vývojovou edicí HoloLens 2 a HoloLens 2?

K vytváření a testování vnitřních podnikových aplikací, které se mají používat s HoloLens 2 (komerční edice), můžete použít jenom vývojovou edici HoloLens 2. V rámci nákupu vývojové edice může společnost Microsoft podle vlastního uvážení poskytovat určitá oprávnění zjištěná během procesu registrace. Tato oprávnění zahrnují právo uplatnit po určitou dobu zvýhodněné kredity nebo zkušební přístup k určitým službám nebo nástrojům Microsoftu nebo jiným výrobcům. [Další informace](hololens2-options-dev-edition.md)

### <a name="can-any-edition-of-hololens-2-be-used-to-develop"></a>Dá se k vývoji použít kterákoli edice HoloLens 2?

Ano. K vývoji aplikací se dá použít libovolná edice HoloLens 2.

### <a name="how-and-will-i-receive-my-500-in-azure-credits-3-month-unity-pro-trial-and-3-month-pixyz-plugin-trial"></a>Jak a se mi dostanou $500 v kreditech Azure, zkušební verze Unity pro a Pixyz vyzkoušení plug-in na 3 měsíce?

Vaše zvýhodněné kredity nebo zkušební verze se budou posílat do e-mailu, který se používá při nákupu do 2 pracovních dnů od dodávky produktu.

### <a name="can-i-upgrade-my-hololens-2-device-only-device-to-the-hololens-2-development-edition-to-get-the-trial-credits"></a>Můžu "upgradovat" moje zařízení HoloLens 2 (jenom zařízení) na vývojovou edici HoloLens 2, abyste získali zkušební kredity?

Ne, nenabízíme plány přenosu zařízení. Přenos z žádné edice do žádné jiné edice není možné.

### <a name="what-is-unity-pro"></a>Co je Unity pro?

Unity pro je 3D platforma v reálném čase pro malé a střední studia/firmy. Je navržená pro týmy se směsí technických a netechnických dovedností. [Další informace](https://store.unity.com/products/unity-pro)

### <a name="what-is-pixyz-plugin"></a>Co je modul plug-in Pixyz?

Modul plug-in Pixyz čte a převádí skoro libovolný formát, optimalizuje a umísťuje ho do Unity a v rámci něj zajišťuje, aby poskytoval prostředky připravené k použití pro aplikace v reálném čase. [Další informace](https://unity.com/products/pixyz)

### <a name="is-financing-for-the-hololens-2-development-edition--available"></a>Je k dispozici financování pro vývojová edice HoloLens 2?

Ano, [financování pro vývojovou edici HoloLens 2](https://ms.liftforward.com/hololens?product=hololens-development-edition) je nyní k dispozici pouze v USA. 

### <a name="who-do-i-contact-with-questions-about-my-hololens-2-development-edition-entitlements"></a>Na koho se mám obrátit s dotazy ohledně vašich nároků na vývojovou edici HoloLens 2?

S případnými dotazy týkajícími se nároků se obraťte na svého odborníka na prodej na 866-425-4709, od až Pá 6:00 do 18:00, PST.

### <a name="can-my-company-buy-multiple-hololens-2-development-edition-devices"></a>Může moje společnost koupit několik zařízení s vývojovou verzí HoloLens 2? 

Toto zařízení je určená pro vývojáře, kteří vytvářejí hybridní řešení realit. Pro každou transakci zákazníka omezujeme jeden nákup. Pokud vaše společnost chce koupit zařízení s více edicemi, bude vyžadovat samostatné transakce k nákupu zařízení s vývojovou verzí HoloLens 2 jménem společnosti. 

### <a name="can-i-show-my-hololens-2-development-edition-device-to-my-customers"></a>Můžu svým zákazníkům zobrazit svoje zařízení HoloLens 2 Development Edition? 

Ano, toto zařízení je určeno k použití při vytváření prototypů a testování hybridních řešení realit.

### <a name="where-can-i-buy-the-hololens-2-development-edition"></a>Kde můžu koupit verzi HoloLens 2 pro vývoj? 

Vývojová edice HoloLens 2 se prodává výhradně na Microsoft Store.  

### <a name="can-i-return-my-hololens-2-development-edition"></a>Můžu se vrátit k vývojové edici HoloLens 2?  

Všechny prodeje jsou konečné a v edici HoloLens 2 pro vývoj nejsou k dispozici žádné refundace.

### <a name="does-developer-edition-have-different-hardware-requirements-that-differ-from-the-hololens-2"></a>Má vývojářská edice různé požadavky na hardware, které se liší od HoloLens 2?

Ne, toto zařízení je stejné jako v perspektivě hardwarových požadavků. Vývojářská edice se může nasadit v případě vývoje a testování aplikací v komerčním nastavení. 

### <a name="how-can-i-use-mixed-reality-toolkit--unity-for-developing-on-my-hololens-2-development-edition"></a>Jak můžu využít smíšené sady realit Toolkit – Unity pro vývoj na vývojové edici HoloLens 2?

MRTK-Unity je projekt založený na Microsoftu, který obsahuje sadu komponent a funkcí urychlujících vývoj aplikací s hybridní realitou v Unity pro různé platformy. [Další informace](https://microsoft.github.io/MixedRealityToolkit-Unity/README.html)

### <a name="how-can-i-use-mixed-reality-toolkit--unreal-uxt-for-developing-on-my-hololens-2-development-edition"></a>Jak můžu použít sadu Mixed reality Toolkit – Unreal UXT pro vývoj na vývojové edici HoloLens 2?

MRTK-Unreal je první komponenta nástrojů pro UX, která se uvolní a je aktuálně podporovaná jenom na HoloLens 2. Modul plug-in komponent obsahuje kód, modrotisky a příklad assetů běžných funkcí uživatelského rozhraní pro vstupní simulaci, objekty actor interakce, komponenty tlačítek, součásti manipulátor a komponenty chování. [Další informace](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)

### <a name="are-os-releases-for-the-hololens-2-development-edition-the-same-as-hololens-2"></a>Jsou verze operačních systémů pro vývojovou edici HoloLens 2 stejné jako HoloLens 2?

Ano.

### <a name="can-i-run-released-sideload-or-store-applications-on-my-hololens-2-development-edition-device"></a>Můžu na svém zařízení HoloLens 2 Development Edition spustit vydané aplikace (bokem nebo Store)?

Pokud implementovat podmínky a ujednání pro vývojovou edici HoloLens 2, můžete spustit jakoukoli vydanou aplikaci v edici Developer Edition. Neomezujeme používání zkušebně načtené nebo uložených aplikací pro tuto SKU.

## <a name="azure"></a>Azure

### <a name="how-can-i-redeem-my-500-azure-credit"></a>Jak můžu uplatnit kredit $500 Azure?

Pokud chcete uplatnit kredity Azure $500, uplatněte Azure Pass propagační kód a aktivujte své předplatné. [Další informace](https://www.microsoftazurepass.com/Home/HowTo?Length=5)

Přečtěte si informace o [smlouvě o předplatném](https://azure.microsoft.com/support/legal/subscription-agreement/?country=US) Azure Pass a [o nabídkách](https://azure.microsoft.com/en-us/offers/azure-pass/).

### <a name="when-does-my-500-azure-credit-expire"></a>Kdy vyprší platnost kreditu na $500 Azure?

Po aktivaci kódu Azure Key vyprší 90 dnů platnost kreditu na $500 Azure.

### <a name="what-can-i-do-with-my-azure-credits"></a>Co se dá dělat s kredity Azure?

Kredity Azure vám umožní začít sestavovat s Microsoft&#39;s neustále se rozšiřující sadou cloudových služeb. Kredity Azure, které jsou součástí vývojové edice HoloLens 2, umožňují zákazníkům rychle začít sestavovat a testovat smíšené aplikace realit v Azure. Úplné podrobnosti o sestavování v Azure najdete v [prostředcích Azure](https://azure.microsoft.com/en-us/resources/) .

### <a name="where-can-i-learn-more-about-azure-mixed-reality-services"></a>Kde se mohu dozvědět více o službách Azure Mixed reality?

Microsoft nabízí sadu smíšených služeb realit, které vývojářům umožňují vytvářet a testovat hybridní řešení realit. [Další informace](https://docs.microsoft.com/windows/mixed-reality/develop/mixed-reality-cloud-services)
