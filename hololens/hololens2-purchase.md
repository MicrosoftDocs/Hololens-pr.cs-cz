# <a name="where-to-buy-hololens-2"></a>kde koupit HoloLens 2

## <a name="overview"></a>Přehled
HoloLens 2 je nejpohodlnější zařízení s hybridní realitou, které nabízí špičková řešení, která dodávají moderní prostředí, která se zvyšují spolehlivostí, zabezpečením a škálovatelností služeb cloud a AI od microsoftu. začněte hned s aplikacemi mimo polici nebo vytvářejte vlastní řešení pomocí Microsoft Azurech služeb.
