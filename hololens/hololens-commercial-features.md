# <a name="hololens-2-capabilities-and-solutions"></a>HoloLens 2 možnosti a řešení

## <a name="why-hololens"></a>Proč HoloLens?

Díky HoloLens 2 a Mixed Reality aplikacím teď mohou pracovníci spolupracovat bez hranic, jednat s přesností a zvyšovat produktivitu.

Spolupracovat bez hranic a jednat s přesností na zvýšení produktivity zaměstnanců pomocí aplikací hybridní reality na HoloLens 2. Díky integrovaným hlasovým příkazům, sledování očí a ukotvení na světě se můžete neustále soustředit na bezpečné dokončení úkolů bez chyb. Připojení se vzdálenými kolegy v reálném čase a spolupracovat na rozsáhlém holografickém plátně překrytém ve fyzickém prostředí a rychle řešit problémy v okamžiku práce. Realizovat návratnost investic (ROI) hned od výrobce s robustním ekosystémem aplikací, které jsou podporovány zabezpečením, spolehlivostí a škálovatelností Microsoftu.  

Doufáme, že vás tento článek inspiruje přemýšlet o všech způsobech, jakými bude hybridní realita pro vaše podnikání přínosem.

## <a name="hololens-2-capabilities"></a>HoloLens 2 možnosti

Proč je HoloLens 2 tak výkonný?

| Funkce | Popis |
|---------|-------------|
| Ukotvení na světě | Ukotvené hologramy zůstanou přesně na místě. HoloLens 2 rozumí vašemu pracovnímu prostoru. Digitální obsah tak v průběhu času přetrvává – ukotvený k objektům nebo povrchům, na kterých pracujete. |
| Ruční sledování | Dotykové ovládání, pochopení a pohyb hologramů způsobem, který se bude cítit přirozený. HoloLens 2 se přizpůsobí vašim rukou, aby se vám nově našly spokojenosti s vašimi interakcemi. |
| Sledování očí | Užívejte si novou úroveň kontextu a porozumění člověku. HoloLens 2 přesně rozumí tomu, kde hledáte, takže dokáže pochopit váš záměr a přizpůsobit hologramy vašim očím v reálném čase. |
| Povolený hlas | Integrované hlasové příkazy umožňují rychle procházet a ovládat HoloLens 2, když jsou ruce zabrané úkolem. |
| Ergonomické | HoloLens 2 je odlehčená (3,28 kg), která obsahuje systém pro dialekt, který podporuje rozšířené použití. |
| Large FoV | Rozbalte holografické plátno s velkými zorné displeji ve vysokém rozlišení. |
| Neomezený | Pohybujte se volně, bez vodičů nebo externích balíčků, abyste se posunout do práce. |
| S využitím Azure | Streamujte vysoce věrný 3D obsah, který lze ukotvit do umístění nebo objektu, který se uchová mezi uživateli pomocí služeb azure hybridní reality.
| Zachycení hybridní reality | Zdokumentovat prostředí jako fotku nebo video, abyste ho v reálném čase sdíleli s ostatními. |
| Windows Hello pro firmy | Biometrické ověřování založené na Iris vás rychle a bezpečně dostane do pracovního procesu. |
| Windows Autopilot | Nastavte a předem nakonfigurujte služby pro HoloLens 2, abyste je mohli hned začít používat v distribuovaných službách. |
| Aktualizace operačního systému | Zajistěte zabezpečení díky měsíčním servisním aktualizacím a využijte nové možnosti produktivity a spravovatelnosti v bi-ročních verzích. |
| Snadná správa zařízení | Spravujte několik HoloLens 2 zařízení současně pomocí řešení, jako jsou Microsoft Intune, VMware Workspace One, MobileIron a mnoho dalších. |
| Provoz v regulovaných prostředích | HoloLens 2 má rozsáhlé portfolio zařízení, které podporuje vysoce regulovaná prostředí, včetně prostředí určených pro třídu ISO 5.0 a třídu UL I, divize 2. |

## <a name="managing-hololens-2-in-your-organization"></a>Správa HoloLens 2 ve vaší organizaci

HoloLens 2 obsahuje funkce, které organizacím usnadňují správu a používání HoloLens zařízení. Některé funkce jsou součástí zařízení, zatímco jiné může povolit [Mobile Správa zařízení (MDM)](hololens-mdm-configure.md) pro [](hololens-provisioning.md) HoloLens nebo prostřednictvím zřizovacího balíčku pomocí Windows [Configuration Designeru.](app-deploy-provisioning-package.md#setup)

| Chci... | Řešení | Description |  
|---------| ------------|------------|
Správa způsobu přihlašování koncových uživatelů | [**Identita**](hololens-identity.md) | HoloLens 2 podporuje několik druhů identit uživatelů – Azure Active Directory (AAD), účet Microsoft (MSA) a místní účty.  |
| Šifrování uživatelských dat | [**Zabezpečení dat**](security-encryption-data-protection.md) | Šifrování dat nástrojem BitLocker je na HoloLens 2 povolené, aby poskytovalo stejnou úroveň ochrany zabezpečení jako ostatní Windows zařízení. | 
Správa HoloLens zařízení v mé organizaci | [**Správa mobilních zařízení**](hololens-mdm-configure.md) | Správa nastavení, výběr aplikací pro instalaci a nastavení konfigurací zabezpečení přizpůsobených potřebám vaší organizace na několika HoloLens 2 zařízeních z centrálního umístění. | 
|Minimalizace doby instalace pro nové uživatele a zařízení | [**Autopilota**](hololens2-autopilot.md) | Nakonfigurujte prostředí pro první použití (OOBE) v Microsoft Endpoint Manager a umožnujte koncovým uživatelům připravovat zařízení pro obchodní použití s malými nebo žádnými interakcemi. |  
| Řízení aktualizací operačního systému pro moje zařízení | [**Windows Aktualizace pro firmy**](hololens-updates.md#managing-updates-by-using-windows-update-for-business) | Windows Update for Business poskytuje řízené aktualizace operačního systému pro zařízení a podporu pro kanál dlouhodobé údržby. |  
| Povolit stahování konkrétních a obchodních aplikací |[**Správa aplikací**](app-deploy-overview.md) | Zvolte, jak distribuovat a řídit aplikace pro vybrané skupiny HoloLens 2 uživatele. | 
| Zobrazení nebo skrytí konkrétních aplikací v nabídce Start |[**Celoobrazovkový režim**](hololens-kiosk.md) | Nakonfigurujte HoloLens 2 tak, aby fungovala jako zařízení s pevným účelem pro použití v ukázkách aplikací nebo vyhrazených obchodních aplikacích. 
| Zabezpečení prostředí uzamknutím aplikací | [**WDAC**](windows-defender-application-control-wdac.md) | Windows Defender Řízení aplikací (WDAC) blokuje spuštění aplikací a procesů uživatelem zařízení.
| Správa zabezpečení zařízení pomocí pravidel pro aplikace a procesy | [**Zásady (CSP)**](hololens-csp-policy-overview.md) | Správci IT mohou definovat a implementovat nastavení zásad pomocí existujícího seznamu podporovaných CSP zásad na HoloLens 2. |  
| Správa způsobu připojení zařízení k internetu | [**Síť a možnosti připojení**](hololens-certificates-network.md) | Pro přístup k Wi-Fi, sítím VPN nebo interním prostředkům použijte ověřování pomocí certifikátů. | 
| Sdílení zařízení s více uživateli | [**Automaticky přizpůsobené zobrazení**](hololens-calibration.md#auto-eye-position-support) | HoloLens 2 se automaticky upraví pomocí funkce AEP (Auto Eye Position), takže při sdílení zařízení mezi uživateli není nutné spouštět ruční proces [úprav.](hololens-multiple-users.md) |

Přečtěte si [o licenčních požadavcích](hololens-licenses-requirements.md) pro výše uvedená řešení.
