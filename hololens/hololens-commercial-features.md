## <a name="hololens-2-capabilities"></a>možnosti HoloLens 2

co přináší HoloLens 2 tak výkonné?

| Funkce | Popis |
|---------|-------------|
| Světová ukotvení | Ukotvené hologramy jsou stále přesně na místě. HoloLens 2 rozumí vašemu pracovnímu prostoru. Digitální obsah tedy přetrvává v čase – ukotvení k objektům nebo povrchům, kde pracujete. |
| Ruční sledování | Dotyková, zajímavý a pohybová hologramy způsobem, který je přirozený. HoloLens 2 se přizpůsobuje vaší práci, aby bylo možné v interakci newfound uspokojivým způsobem. |
| Sledování očí | Využijte novou úroveň kontextu a lidského porozumění. HoloLens 2 se přesně seznámí s tím, kde hledáte, takže může pochopit svůj záměr a přizpůsobovat hologramy na oči v reálném čase. |
| Hlas povolen | integrované hlasové příkazy umožňují rychlou navigaci a provozování HoloLens 2, pokud se s úlohou pracujete na ruce. |
| Ergonomick | HoloLens 2 je odlehčená (3,28 kg), která zahrnuje systém přizpůsobený vytáčením, který podporuje rozšířené použití. |
| Velké FoV | Rozšiřte své holografické plátno s vysokým rozlišením a zobrazeními velkých polí. |
| Nepřipojeno | Pohybujte volně, a to bez vodičů nebo externích sad, abyste mohli na úlohu získat svůj způsob. |
| Azure – napájené | Datový proud s vysokou věrností 3D obsahem, který se dá ukotvit k umístění nebo objektu, který se v různých uživatelích uchovává se službami Azure Mixed reality.
| Zachycení směsné reality | Zdokumentujte si prostředí jako fotografii nebo video, které chcete sdílet s ostatními v reálném čase. |
| Windows Hello pro firmy | Biometrické ověřování založené na Iris vám umožní rychle a bezpečně pracovat s tokem práce. |
| Windows Autopilot | nastavte a předem nakonfigurujte služby pro HoloLens 2, aby byly připravené k použití přímo v distribuovaných worksites. |
| Aktualizace operačního systému | Udržujte si přehled o měsíčních servisních aktualizacích a využijte výhod nových funkcí pro produktivitu a možnosti správy v rámci obchodních verzí. |
| Snadná správa zařízení | spravujte víc zařízení HoloLens 2 současně pomocí řešení, jako je Microsoft Intune, pracovního prostoru VMware One, MobileIron a spousta dalších. |
| Provoz v regulovaných prostředích | HoloLens 2 má rozsáhlé portfolio zařízení, které podporuje vysoce regulovaná prostředí včetně prostředí určených pro ISO class 5,0 a UL třídy I, divize 2. |

## <a name="managing-hololens-2-in-your-organization"></a>správa HoloLens 2 ve vaší organizaci

HoloLens 2 obsahuje funkce, které organizacím usnadňují správu a používání HoloLensch zařízení. některé funkce jsou součástí zařízení, zatímco jiné můžou povolit [správa mobilních zařízení (MDM) pro HoloLens](hololens-mdm-configure.md) nebo prostřednictvím [zřizovacích balíčků](hololens-provisioning.md) pomocí [Windowsho návrháře konfigurace](app-deploy-provisioning-package.md#setup).

| Chci... | Řešení | Description |  
|---------| ------------|------------|
Správa způsobu přihlášení mých koncových uživatelů | [**Identita**](hololens-identity.md) | HoloLens 2 podporuje několik druhů identit uživatelů – Azure Active Directory (AAD), účet Microsoft (MSA) a místní účty.  |
| Šifrování uživatelských dat | [**Zabezpečení dat**](security-encryption-data-protection.md) | šifrování dat nástroje BitLocker je na HoloLens 2 povoleno, aby poskytovalo stejnou úroveň ochrany zabezpečení jako jakékoli jiné zařízení Windows. | 
správa zařízení HoloLens v mojí organizaci | [**Správa mobilních zařízení**](hololens-mdm-configure.md) | spravujte nastavení, vyberte aplikace, které chcete nainstalovat, a nastavte konfigurace zabezpečení přizpůsobené potřebám vaší organizace na více zařízení HoloLens 2 z centrálního umístění. | 
|Minimalizace času nastavení pro nové uživatele a zařízení | [**Autopilot**](hololens2-autopilot.md) | nakonfigurujte prostředí spouštěné při prvním zapnutí (OOBE) v Microsoft Endpoint Manager a umožněte koncovým uživatelům připravovat zařízení pro podnikové použití bez jakýchkoli interakcí. |  
| Řízení aktualizací operačního systému pro moje zařízení | [**Windows Aktualizace pro firmy**](hololens-updates.md#managing-updates-by-using-windows-update-for-business) | Windows Aktualizace pro firmy poskytuje řízené aktualizace operačního systému pro zařízení a podporu pro kanál pro dlouhodobé obsluhy. |  
| Umožňuje stažení konkrétních a obchodních aplikací. |[**Správa aplikací**](app-deploy-overview.md) | vyberte způsob distribuce a řízení aplikací pro vybrané skupiny uživatelů HoloLens 2. | 
| Zobrazit nebo skrýt konkrétní aplikace v nabídce Start |[**Celoobrazovkový režim**](hololens-kiosk.md) | nakonfigurujte HoloLens 2 tak, aby fungovaly jako zařízení s pevným účelem pro použití v ukázkách aplikací nebo vyhrazených obchodních aplikacích. 
| Zabezpečení mého prostředí uzamčením aplikací | [**WDAC**](windows-defender-application-control-wdac.md) | Windows Defender Řízení aplikací (WDAC) blokuje spouštění aplikací a procesů uživatelem zařízení.
| Správa zabezpečení zařízení pomocí pravidel pro aplikace a procesy | [**Zásady (CSP)**](hololens-csp-policy-overview.md) | správci IT můžou definovat a implementovat nastavení zásad pomocí existujícího seznamu podporovaných csp zásad na HoloLens 2. |  
| Správa způsobu připojení zařízení k Internetu | [**Síť a připojení**](hololens-certificates-network.md) | Použijte ověřování založené na certifikátech pro přístup k Wi-Fi, VPN nebo interním prostředkům. | 
| Sdílet zařízení s více uživateli | [**Automatické přizpůsobené zobrazení**](hololens-calibration.md#auto-eye-position-support) | HoloLens 2 se automaticky upravují pomocí pozice automatického oka (AEP), což eliminuje nutnost spuštění ručního procesu kalibrace při [sdílení zařízení mezi uživateli](hololens-multiple-users.md). |

Seznamte se s [požadavky na licencování](hololens-licenses-requirements.md) pro výše uvedená řešení.
